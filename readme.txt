=== MaVo GeoTag Plus ===
Requires at least: 6.0
Requires PHP: 7.4
License: GPLv2

Automatically adds multilingual geographic tags (continent, country, region, county, city)
to WordPress posts that have a Geo Mashup location defined.

== Requirements ==
* Geo Mashup >= 1.9
* Polylang (free)

== Usage ==
Install and activate. The plugin tags posts automatically on save.
Use Tools → Geo Tagger to run the batch processor on existing posts.
